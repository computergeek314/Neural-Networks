Maze Explorer

This is a maze exploration 3d demo I created in C and compiled for my Ti-89 calculator using the GCC4TI cross compiler. A random maze with sliding doors that you open is created for the user to explore. Pressing MODE shows the user a map of parts of the maze they have explored.


The main purpose of this application was not to entertain but rather to demonstrate the capabilities of a graphics engine I am developing. The raycasting engine can render perpendicular walls with arbitrary textures (even textures with transparency). Since the calculator lacks floating point hardware, this is done using only fixed point arithmetic with 16 bit integers, allowing the demo to run at a solid 10 frames per second.


See GCC4TI for info on how to create your own calculator assembly games!
