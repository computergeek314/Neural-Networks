Apple 1 Emulator

This is a sudoku game I created in C and compiled for my Ti-89 calculator using the GCC4TI cross compiler. The game can generate random sudoku puzzles which vary in difficulty and are guaranteed to have only 1 solution. It features a pleasing note-taking ui as well as the ability to save and load incomplete puzzles.


See GCC4TI for info on how to create your own calculator assembly games!
