QR Code Generator

QR-util is a fast and simple command-line qr code generator. It supports generating all 40 sizes of QR codes, all 4 error correction levels, and all 8 masks. It will automatically choose the size, error correction level, and mask for the user; however, it also allows the user to specify each of these. It outputs a bitmap which can be optionally upscaled or inverted. Unlike most other QR code generators, this project is open source and licensed with the MIT license, which allows others to freely use it in their software.
