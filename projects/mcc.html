Minimal C Compiler

This is my first C compiler written in C. It compiles a subset of C to MIPS assembly which runs in SPIM and MARS simulators. Compilation is done in a single pass, making it very fast (although it does not generate very efficient assembly). Ultimately the goal is to port the compiler onto my calculator so that I can write, compile, and run C programs on my calculator!
